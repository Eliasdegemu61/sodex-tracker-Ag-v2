# Sodex Tracker

A minimalist, high-fidelity dashboard for tracking on-chain activity, portfolios, and trade analytics on the SoDex platform.

## Features
- **Wallet Tracker**: Real-time position and balance monitoring.
- **Portfolio Analytics**: Deep dive into trade history and performance metrics.
- **Community Pulse**: Social sentiment and activity tracking.
- **Leaderboard**: Track top performers across the SoDex ecosystem.

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- Supabase
- Recharts
